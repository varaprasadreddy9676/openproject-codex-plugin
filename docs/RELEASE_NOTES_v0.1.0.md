# v0.1.0

First public release of `openproject-codex`.

## Highlights

- OpenProject project, user, group, role, membership, and query tooling
- work package CRUD, comments, relations, watchers, attachments, and file links
- bulk work package and membership operations
- boards, wiki pages, and meetings from Codex
- time entries, documents, and news support
- reporting layer for workload, burndown, overdue, and project health export
- HTML and PNG dashboard export

## Why this release matters

OpenProject's official MCP surface is still limited for many writable workflows.

This plugin gives Codex a practical OpenProject operating surface now, including:

- project coordination
- task execution
- bulk operations
- content updates
- reporting
