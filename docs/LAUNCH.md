# Launch Kit

Use this file when announcing `openproject-codex`.

## Positioning

`openproject-codex` is for teams that want to operate OpenProject from Codex instead of treating Codex as read-only.

It supports:

- project and team management
- work package CRUD
- bulk task workflows
- boards, wiki, and meetings
- attachment upload
- reporting and dashboard export

## Medium launch post

Released `openproject-codex`, an open-source Codex plugin for OpenProject.

The goal is simple: do OpenProject work from Codex instead of bouncing back to the OpenProject UI.

Current support includes:

- projects, memberships, users, groups, roles
- work package create/update/delete
- bulk update/comment/watch/delete workflows
- boards, wiki pages, and meetings
- binary attachment upload
- workload, burndown, overdue, and health reporting
- HTML and PNG dashboard export

Repo:

- `https://github.com/varaprasadreddy9676/openproject-codex-plugin`
